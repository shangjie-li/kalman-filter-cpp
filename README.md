# kalman-filter-cpp

Implementation of multiple target tracking based on kalman filter in C++

## 安装
 - 编译并生成可执行文件
   ```
   mkdir build
   cd build
   cmake .. # 生成供make编译的规则文件makefile
   make # 编译源文件并生成可执行文件multiple_target_tracking
   ```

## 运行
 - 启动多目标跟踪模拟程序
   ```
   ./multiple_target_tracking
   ```
